  Web Tabanlı Programlama

Dersin Amacı İnternet, WWW,sunucu-istemci kavramlarını Visual Studio 2010 Geliştirme ortamının kullanımı Visual Studio 2010 IDE ‘ de yer alan standart ve gelişmiş kontrollerin kullanımını 3 Katmanlı mimariye uygun web tabanlı programlar geliştirmeyi öğrenmek  
Dersin Öğrenme Çıktıları
1-İnternet, WWW, HTML dili, sunucu-istemci mimarisi, sunucu tabanlı diller hakkında bilgi sahibi olmak  
2-Visual Studio 2010 IDE ortamını kullanarak Asp.Net kontrollerinin kullanım becerisi kazanmak 
3-Web tabanlı yazılım geliştirme becerisi kazanmak 
4-C# programlama dilini kullanma becerisi kazanmak 
5-3 Katmanlı mimaride web tabanlı yazılımlar geliştirme 
Dersin İçeriği Derse; internet, istemci sunucu mimarisi, haberleşme protokollerinin tanımlanması, HTML tagları ve kullanımlarının anlatılması ile başlanmaktadır. Sunucu tabanlı diller tanıtılmakta ve ASP.NET ile web tabanlı programlar geliştirmekte kullanılan Visual Studio 2010 Geliştirme ortamının kullanımı, web projelerinin geliştirilmesi sırasında kullanılan kontroller ve kullanımları ayrıntılı olarak anlatılmaktadır. Ayrıca anlatım içersinde 3 Katmanlı mimari kullanarak, Microsoft SQL Server veritabanı sunucusu üzerinde veri depolayan örnek uygulama adım adım kodlanmakta ve test edilmektedir.
